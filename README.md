# smoothy

A React 19 + GSAP Flip library for two-state shadcn Card morphs. Wrap any two layouts in `<Smoothy.Card state="a" | "b">` and tag only shared elements with `data-smoothy-id`. smoothy handles shared-element morphing for the card + shared nodes and progressively reveals non-shared content with a strong blur-to-sharp transition on both expand and collapse. See [packages/smoothy/README.md](./packages/smoothy/README.md) for installation and usage.
